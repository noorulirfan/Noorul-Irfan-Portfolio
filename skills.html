HTML
HTML5 is the most recent version of the HTML standard, which is a markup language used to bring structure to online content. Web developers use HTML5 to determine how to structure text like paragraphs, headers, lists, links, and other element categories..
JAVA SCRIPT
JavaScript is the primary programming language for interactive web pages and is utilized on most current websites. Web developers use JavaScript to bring websites to “life” by controlling actions such as dynamic loading. This language also specifies an element’s reaction to a user’s click, thus enabling an infinite number of interactive options..
CSS
Web developers use CSS to manage the overall appearance of site components across many pages. CSS language may affect the appearance of font faces, colors, element spacing, and element sizes.
